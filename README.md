# Soonbway
> 순천향대학교 학생들을 위한 실시간 1호선 위치 제공 앱

## 지원기능
- 신창역을 기준으로 출발/도착 1호선 위치 제공
- 실시간 서비스 제공
- 바로가기 버튼

## 지원 예정
- 학교도착까지 최소소요시간 지하철-버스 연동
- 신창역 이외의 지하철 선택
